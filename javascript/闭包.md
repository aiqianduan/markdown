## 闭包

> 什么是闭包?
>
> 1. 密闭的容器,类似于set,map容器,用于存储数据
> 2. 闭包是一个对象,存储数据的格式: key:value
>
> 闭包形成条件
>
> 1. 函数嵌套
> 2. 内部函数引用外部函数的局部变量
>
> 闭包的优缺点
>
> - 优点: 延长外部函数局部变量的生命周期
> - 缺点:容易造成内存泄漏
>
> 注意点:
>
> 	1. 合理的使用闭包
>  	2. 用完闭包及时清除

```js
function fun() {
    var count = 1;
    return (() => {
        count++
        console.log(count)
    })
}
var f1 = fun()
f1() //2
f1() //3
```
