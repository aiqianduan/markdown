###### post请求发送数据:
- form表单提交:Content-type:application/x-www-form-urlencoded
- file upload文件上传:Content-type:multipart/form-data
- fetch/axios上传:Content-type:application/json

###### HTTP方法
- GET       读取
- POST      发布
- PUT/PATCH 更新
- DELETE    删除

###### restful api
- GET/users 读取所有用户
- GET/users/{id} 读取某个用户
- POST/users 添加一个用户
- PUT/PATCH/users/{id} 修改某个用户
- DELETE/users/{id} 删除某个用户
